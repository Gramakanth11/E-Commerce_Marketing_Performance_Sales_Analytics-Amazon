# E-Commerce Marketing Performance Sales Analytics-Amazon

### Project Overview

This project analyzes e-commerce transaction data to understand sales performance, marketing impact, and customer behavior. The analysis combines Python, SQL, and Power BI to build an interactive business intelligence dashboard.

### Dataset

~50,000 transaction records

Product categories, discounts, regions, payment methods

Revenue and sales metrics

### Project Objectives

This project aims to:

Analyze sales performance across regions and product categories

Identify top-selling products

Evaluate the impact of discounts on sales

Understand payment preferences and customer behavior

Track monthly revenue trends

### Tools Used

Python (Pandas)

MySQL

Power BI

### Project Workflow

Raw Dataset -> Data Cleaning (Python) -> Data Storage (MySQL) -> SQL Analysis -> Power BI Dashboard -> Business Insights

### Key Insights

Generated $32.87M total revenue from 50K orders

Electronics category drives the highest sales volume

Revenue distribution is balanced across North America, Middle East, Asia, and Europe

Discounts show a positive correlation with sales volume

Sales peaks occur during August and October

### Dashboard Features

Revenue by category and region

Top-selling products

Payment method distribution

Discount impact analysis

Monthly revenue trend

### Business Recommendations

Increase promotions for electronics category

Run targeted campaigns during peak sales months

Use discounts strategically to boost demand

Encourage digital payments through incentives
